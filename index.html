Hexo基本命令基本命令 1.hexo init:用于初始化一个本地文件夹为网站的根目录 2.hexo new title:新建一篇文章，会在source文件夹下创建一个以title为文件名的md文件，对该文件进行编辑 3.hexo generate 或 hexo g:用于生成静态文件 4.hexo server 或 hexo s:用于启动本地服务器 5.hexo deploy 或 hexo d:用于部署网站，将静态文件上传到github上 6.hexo clean:用于清理缓存文件 部署流程hexo clean && hexo g && hexo d
