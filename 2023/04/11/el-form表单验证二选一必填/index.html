自定义表单验证规则

若要使表单验证实现两个el-form-item必选其中一项，可以在表单验证中为特定的项设置validator，例如

validator属性的值为一个函数function(rule, value, callback, source, options)

  • rule:与正在验证的字段名称相对应的源描述符中的验证规则。
  • value:源对象中被验证的值
  • callback:验证完成后调用的回调函数。它期望传递一个 Error 实例数组以指示验证失败。如果检查是同步的，你可以直接返回一个 false 或 Error 或 Error Array。
